# Timex – All-in-One Time & Utility Calculator App

Timex is a multi-functional Android application designed to perform advanced time calculations, financial estimations, and daily utility conversions in a single offline platform.

 ## Features

✅ Time Calculator (Addition & Subtraction)  
✅ Age Calculator  
✅ Duration Calculator  
✅ Time-Based Earnings Calculator  
✅ Simple Interest Calculator  
✅ Unit Converter  
✅ Currency Converter  
✅ Fuel Cost Estimator  
✅ Number to Words Converter  
✅ Notes Counter  

## Key Highlights

- Fully Offline Android Application
- Fast and Accurate Calculations
- Clean Card-Based User Interface
- Modular Feature Architecture
- User-Friendly Navigation
- Lightweight & Performance Optimized

## Tech Stack

- Language:** Java  
- UI Design:** XML
